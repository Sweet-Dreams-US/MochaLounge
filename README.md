# Mocha Lounge — Demo Website

Static demo concept for **Mocha Lounge**, a Fort Wayne neighborhood coffee lounge with two locations (Covington Rd and Dupont Rd). Built by Sweet Dreams Studios.

## Pages

- `index.html` — home
- `menu.html` — full menu
- `story.html` — origin & timeline
- `visit.html` — both locations, hours, contact form

## Stack

Pure static — HTML + CSS + a tiny vanilla JS file. No build step. Hosted free on GitHub Pages.

- **Type:** Bricolage Grotesque, Fraunces (italic accents), Inter
- **Palette:** Mocha ink `#15100D`, latte cream `#FAF4EC`, honey-amber `#D89B4A`, sage `#7B8E6E`
- **Imagery:** Unsplash placeholders — swap with real photography when ready

## Local preview

Just open `index.html` in a browser, or run a quick static server:

```bash
python -m http.server 8000
# or
npx serve .
```

## Disclaimer

This is a demo concept. Menu prices, photos, and some narrative details are illustrative — Mocha Lounge would supply finals before launch.
